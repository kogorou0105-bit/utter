# Scenarios

This folder is for real conversation situations.

Each note should answer one question: if this happens again, how can I respond naturally in English?

## Categories

- `emotional-support/`: comforting, encouraging, showing care
- `small-talk/`: casual topics and daily conversation
- `dating-and-social/`: social, romantic, and friendly conversations
- `work-and-study/`: school, work, interviews, collaboration

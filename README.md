# utter

This repository is my personal English speaking lab.

The goal is not to memorize random words, but to build a practical expression system for real conversations.

## Principles

1. Start from real situations.
2. Record what I wanted to say in Chinese.
3. Find natural English versions.
4. Compare similar words and expressions.
5. Practice output regularly.

## Workflow

1. When I get stuck in a conversation, write the Chinese idea in `00-inbox/raw-thoughts.md`.
2. Later, turn it into a scenario note under `01-scenarios/`.
3. Extract useful expressions into `02-phrases/`.
4. Extract useful words and word differences into `03-vocabulary/`.
5. Practice output in `05-drills/`.

## Weekly Routine

1. Pick 3 real situations from `00-inbox/`.
2. Rewrite each situation in natural English.
3. Choose 10 expressions I want to own.
4. Say each expression out loud in 3 different contexts.
5. Review old notes and mark expressions I actually used.
